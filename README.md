# Api Playground App

This is a simple application to test and explore APIs. 

## Feature list
* Request composer
* Pipeline stages
* Timeout/cancel
* Response display

### Note

Backend is mocked for support of AbortSignal.

## How to run it locally

### Installation

Install the dependencies:

```bash
npm install
```

### Development

Start the development server with HMR:

```bash
npm run dev
```

Check for type and lint errors:

```bash
npm run typecheck
npm run lint
```
